¿Cómo consigo mi token? (solo la primera vez)
  1. Entra a github.com/settings/tokens (inicia sesión con tu cuenta).
  2. Clic en Generate new token.
  3. En Repository access elige Only select repositories → selecciona menu-abejita.
  4. En Permissions → Repository permissions → Contents → ponlo en Read and write.
  5. Clic en Generate token, copia el texto que aparece y pégalo arriba.

El token queda guardado solo en este dispositivo. Si lo pierdes o quieres cambiarlo, puedes revocarlo en GitHub y crear otro.
